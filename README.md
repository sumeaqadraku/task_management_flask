Task Manager API
A simple Flask-based REST API for managing tasks. This application allows you to create, read, update, and delete tasks efficiently.

<br>
Features
List Tasks: Retrieve all tasks.
Add Task: Create a new task.
Update Task: Modify an existing task.
Delete Task: Remove a task.
